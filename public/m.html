1. Introduction

Le projet « Plateforme Étudiante – Dakar » vise à centraliser toutes les informations utiles pour les étudiants vivant à Dakar, au Sénégal. Il s'agit d'une solution numérique moderne permettant de faciliter la vie étudiante : recherche de logement, informations sur les écoles, entraide, revente d'objets, etc.

🎯

2. Objectifs du projet

🏠 Point d'accès unique

Centraliser toutes les ressources nécessaires à la vie étudiante à Dakar

🤝 Entraide communautaire

Favoriser l'entraide et la solidarité entre étudiants

🔍 Recherche simplifiée

Faciliter la recherche de logement et d'établissements

💱 Marketplace étudiant

Permettre la revente et l'achat de matériel étudiant

⚙️

4. Architecture technique

🔧 Backend

Laravel 10+, PostgreSQL, API REST sécurisée

💻 Frontend Web

React + Vite, Tailwind CSS, design responsive

📱 Mobile

Flutter multiplateforme, même API

⚡ Admin

Filament Admin pour la gestion

🗺️ Cartographie

Leaflet.js (web), Google Maps (mobile)

📁 Fichiers

Spatie Media Library

📊
